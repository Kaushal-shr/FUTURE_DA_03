# Student Satisfaction Feedback Analysis

## Project Overview
This project analyzes student satisfaction survey data collected from academic courses. The objective is to evaluate overall satisfaction levels, identify high- and low-performing courses, and provide data-driven insights to support academic improvement initiatives.

## Dataset
- Student Satisfaction Survey (CSV format)
- Contains course-wise feedback ratings on a 1â€“5 scale

## Tools & Technologies
- Python
- pandas
- matplotlib
- Google Colab

## Key Findings
- The overall average student satisfaction rating is approximately 3.84, indicating moderate to high satisfaction.
- Several courses consistently achieve high satisfaction scores, while a few show significantly lower ratings.
- Satisfaction levels vary across courses, highlighting the importance of targeted evaluation.

## Conclusion
The analysis demonstrates how survey data can be transformed into actionable insights to support decision-making in academic environments.
